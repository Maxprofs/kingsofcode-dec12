CSS Exclusions

Shape Inside


    /* wrap the text inside a circle */
    #content { 
        shape-inside: circle(50px, 50px 50px)
    }

[ILLUSTRATION]

Shape Outside


    #shape { 
        shape-outside: circle(50px, 50px 50px)
    }
[ILLUSTATION]

Define an Exclusion


    /* wrap the surrounding text around a circle */
    #exclusion { 
        shape-outside: circle(50px, 50px 50px)
        
        /* text content goes around me */
        wrap-flow: both;
    }

More shapes


    /* rounded rectangle shapes */
    #content { 
        shape-inside: rectangle(x, y, width, height, rx, ry)
    }
    
    /* non-rectangular shapes */
    #content { 
        shape-inside: polygon(x1, y1 x2, y2 ... )
    }

Shapes from SVG


    <svg ...>
        <circle id="circle_shape" cx="50%" cy="50%" r="50%" />
    </svg>

    #content { 
        shape-inside: url(#circle_shape)
    }
